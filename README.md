## Customization Guide
- **Logo**: Replace `assets/images/logo.png`
- **Hero Background**: Replace `assets/images/bg.jpeg`
- **Service Images**: Replace `assets/images/hair1.jpg`, `assets/images/hair2.jpeg`, `assets/images/hair3.jpeg`
- **Team Photos**: Replace `assets/images/stylist.jpeg`, `assets/images/male.jpeg`
- **Colors & Fonts**: Edit `assets/css/style.css`
